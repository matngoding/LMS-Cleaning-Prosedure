# Setup Instructions for LMS-Cleaning-Prosedure

## Installation and Setup Instructions

### Backend Development
1. **Clone the Repository**  
   Open your terminal and run:  
   ```bash  
   git clone https://github.com/matngoding/LMS-Cleaning-Prosedure.git  
   ```  

2. **Navigate to the Backend Directory**  
   ```bash  
   cd LMS-Cleaning-Prosedure/backend  
   ```  

3. **Install Dependencies**  
   Make sure you have Node.js and npm installed, then run:  
   ```bash  
   npm install  
   ```  

4. **Run Database Migrations**  
   ```bash  
   npm run migrate  
   ```  

5. **Start the Server**  
   ```bash  
   npm start  
   ```  

### Frontend Development
1. **Navigate to the Frontend Directory**  
   ```bash  
   cd ../frontend  
   ```  

2. **Install Dependencies**  
   Make sure you have Node.js and npm installed, then run:  
   ```bash  
   npm install  
   ```  

3. **Start the Development Server**  
   ```bash  
   npm start  
   ```  

### Additional Information
- Make sure to set up your environment variables as needed for the backend.  
- Refer to the individual service documentation for specific instructions on functionality and features.
